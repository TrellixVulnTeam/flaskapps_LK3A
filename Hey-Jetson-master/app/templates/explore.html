Automatic Speech Recognition Inference

Introduction

This website runs a live demo of a scalable automatic speech recognition platform on an Nvidia Jetson built with a neural network for speech inference that was inspired by my mental health career with a cosine similarity of about 80% in the validation set and 78% in the test set and word error rate of around 16% in the validation set and 18% in the test set.
