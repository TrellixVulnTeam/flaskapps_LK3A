Resources

All following examples use the requests module of python, though any method of get/put on the URL will trigger the resource. To use the following examples in a python project, you will need to include:

from requests import get, put

UndockRobot

'/api/undock'

Undock robot from charger

get('http://localhost:5000/api/undock').json()

DockRobot

'/api/dock'

Dock robot on charger

get('http://localhost:5000/api/dock').json()

ConnectCube

'/api/connect_cube'

Connect to cube

get('http://localhost:5000/api/connect_cube').json()

DockCube

'/api/dock_cube'

Dock cube and pick it up

get('http://localhost:5000/api/dock_cube').json()

GetStatus

'/api/status'

Get the data from the Status table

get('http://localhost:5000/api/status').json()

AddCommand

'/api/add_command'

Get the data from the Command table

get('http://localhost:5000/api/add_command').json()

Add a command to the Command table

put('http://localhost:5000/api/add_command', data={'data': 'robot.behavior.drive_off_charger()'}).json()

ExecuteCommands

'/api/execute'

Execute all commands in the Command table

get('http://localhost:5000/api/execute').json()

ClearCommands

'/api/clear_commands'

Clear all commands in the Command table

get('http://localhost:5000/api/clear_commands').json()

RunApplication

'/api/run/app_name'

If application Hello World is installed,

you can run it with:

get('http://localhost:5000/api/run/Hello-World').json()

*Note: All dashes are replaced with spaces. Hello-World will be Hello World

ModRunApplication

'/api/modrun/app_name/option=value%'

If application Display Text is installed,

you can modify its settings file and run it with:

get('http://localhost:5000/api/modrun/Display-Text/text_to_show=VectorCloud%font_size=20%').json()

*Note: All dashes are replaced with spaces. Hello-World will be Hello World

ApplicationSettings

'/api/appset/app_name/option=value%'

If application Display Text is installed,

you can modify its settings file with:

get('http://localhost:5000/api/appset/Display-Text/text_to_show=VectorCloud%font_size=20%').json()

*Note: All dashes are replaced with spaces. Hello-World will be Hello World

GetVectors

'/api/vectors'

Get all Vectors in the Vectors table

get('http://localhost:5000/api/vectors').json()

ServerShutdown

'/api/server_shutdown'

Shutdown the computer running VectorCloud

get('http://localhost:5000/api/server_shutdown').json()

ServerReboot

'/api/server_reboot'

Reboot the computer running VectorCloud

get('http://localhost:5000/api/server_reboot').json()
