# super-bbs
一个基于Flask的bbs论坛类项目
前端有用户和管理员两套界面

生产环境启动命令:

`gunicorn -w 8 -k gevent  --log-level warning -b 0.0.0.0:8000 prod_run:app`


### 声明： 严重高仿(照抄)V2EX
### 开发原因： 前后端分离，替换原来的FakeV2EX项目
